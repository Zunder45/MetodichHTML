Главная страница и боковое меню

Кнопка ">>" открывает боковое меню с выбором страниц.

Кнопка "Светлый" на странице методичке меняет тему с светлой на темную или на оборот.

Кнопка "Начальная страница" в меню открывает страницу с темой методички (титуль). (pages/pages/0.html)

Cтраницы Страница - это созданный по шаблону html файл лежащий в каталоге pages\pages. Что бы страница отображалась на сайте нужно:
  • Создать в каталоге pages\pages html файл и скопировать в него шаблон
  • Вносим изменения в документ
  • Создаем кнопку в документе index.html
  • Переходим на сайт
  • Открываем боковое меню
  • Нажимаем на кнопку
Кнопоки

Создание кнопки происходть в документе index.html c помощью тега <btt></btt>

<btt title="Название файла страници (без расширения)">Название кнопки</btt>

Раздел в боковом неню

<z title="Раздел">*Содержание раздела*</z>

Ссылоки

<res title="Заголовок"> Название файла (с расширением (.txt .html и тд))</res>

Создание методички
  • Заполняется тема методички в титульной странице (/pages/pages/0.html)
  • Создаем страницы в /pages/pages
  • Создаем кнопкуи
  • Создаем разделы методички
Теги страниц
Заголовок

<hp>Заголовок</hp>

Картинка

<imag title="ТЕКСТ ПОД КАРТИНКОЙ">НАЗВАНИЕ КАРТИНКИ</imag>

Картинки берутся из каталога pages/src/img

Цветной текст

<c title="green">цвет</c>

Автотабуляция Автотабуляция выключается в файле pages/js/scr.js
Файлы и каталоги

Файлы и каталоги пользователя:

/Pages/Pages Каталог в каторой хранятся страницы методички
/Pages/Pages/0.html Начальная страница (титульник) (можно изменит в index.html)
/Pages/Pages/шаблон.html Шаблон для страницы методички
/Pages/src/file Каталог, где хранятся файлы методички (используется для тега res)
/Pages/src/img Каталог, где хранятся изображения (скриншоты, схемы и тд) методички (используется для тега imag)
